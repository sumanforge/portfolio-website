# Suman Das | Portfolio Website

This is a modern personal portfolio website built with HTML, CSS, and JavaScript. It showcases my background, technical skills, selected projects, and contact information in a clean and interactive experience.

## Features
- Responsive layout for desktop and mobile devices
- Smooth scrolling and animated UI elements
- Light/Dark theme toggle
- Project showcase section
- Contact form section with interactive status feedback

## Technologies Used
- HTML5
- CSS3
- JavaScript
- Font Awesome Icons
- Google Fonts

## Project Structure
- index.html - Main structure of the website
- styles.css - Styling and responsive design
- script.js - Interactivity, theme toggle, and form behavior
- images/ - Images and assets used in the portfolio

## Run Locally
1. Open the project folder in your browser, or use a local server such as VS Code Live Server.
2. Open index.html to view the website.

## Deployment
This portfolio is designed to be deployed easily on GitHub Pages.
